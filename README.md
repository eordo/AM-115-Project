# Stochastic User Equilibrium and System Optimum in Congested Transportation Networks

This is my final project for AM 115 "Mathematical Modeling."

## Setup

Create the project Conda environment:

```bash
conda env create -f environment.yml
conda activate am115_project
```

Then the main results can be replicated by executing `project.ipynb`.
